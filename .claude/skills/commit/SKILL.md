# Skill: commit

Создаёт git-коммит для текущих изменений в репозитории `my_happy_kids`.
Используй этот скил когда пользователь просит «закоммить», «сохрани изменения», «сделай коммит».

## Разрешения

```yaml
tools:
  exec: allow
```

## Шаги

### 1. Проверить статус

```bash
git status
git diff --stat
```

Если `git status` показывает `nothing to commit` — сообщить пользователю и остановиться.

### 2. Изучить изменения

```bash
git diff
git diff --cached
```

На основе diff составить сообщение коммита:
- Короткий заголовок (до 72 символов), **императив** на английском: `Add`, `Fix`, `Update`, `Remove`, `Refactor`, `Docs`
- Описывает **что сделано** (action), не как
- Примеры из истории проекта: `Add RU/EN multilingual support`, `Move language switcher from header to Settings`, `Add 'Clear database' button in General settings tab`
- Если изменений несколько разных файлов — одна обобщающая фраза

### 3. Подтвердить сообщение коммита

Показать пользователю сгенерированное сообщение и спросить через `ask_user_question`:

- Вопрос: «Сообщение коммита: `<сгенерированное сообщение>`. Использовать?»
- Варианты: **«Да, подходит»** / **«Изменить»**

Если пользователь выбрал **«Изменить»** — использовать его текст из поля `custom_text` (вариант Other) как новое сообщение коммита. Если пользователь ввёл пустую строку — переспросить.

### 4. Добавить файлы и закоммитить

```bash
git add -A
git commit -m "$(cat <<'EOF'
<итоговое сообщение коммита>

Generated with [Devin](https://cli.devin.ai/docs)

Co-Authored-By: Devin <158243242+devin-ai-integration[bot]@users.noreply.github.com>
EOF
)"
```

**Важно:** Не использовать `-i` (interactive). Не пушить.

### 5. Отчёт

Показать пользователю:
- Хэш коммита (первые 7 символов)
- Сообщение коммита
- Краткую статистику (`git show --stat HEAD`)

## Правила

- Никогда не коммитить `.env`, `data.db`, `tools/cloudflared.exe`, `cookies.txt` (они в `.gitignore`, но перепроверить)
- Не менять git config
- Не пушить без явной просьбы
